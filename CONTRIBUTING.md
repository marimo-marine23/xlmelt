# Contributing to xlmelt

xlmelt への貢献に興味を持っていただき、ありがとうございます！

## 貢献の方法

### バグ報告

[GitHub Issues](https://github.com/marimo-marine23/xlmelt/issues) からバグを報告してください。

報告時に以下の情報を含めてください:
- xlmelt のバージョン (`xlmelt --version`)
- Python のバージョン
- OS
- 再現手順
- 期待する動作と実際の動作

**注意**: 機密情報を含む Excel ファイルは添付しないでください。問題を再現できる最小限のサンプルファイルを作成してください。

### 機能提案

新機能の提案は Issue で受け付けています。以下を記載してください:
- ユースケース（どのような場面で必要か）
- 期待する動作
- 可能であれば、入力と出力の具体例

### プルリクエスト

1. リポジトリを Fork する
2. フィーチャーブランチを作成する (`git checkout -b feature/my-feature`)
3. 変更をコミットする
4. テストを実行して通ることを確認する (`pytest tests/ -v`)
5. PR を作成する

#### PR のガイドライン

- 既存のテストが通ることを確認してください
- 新機能には対応するテストを追加してください
- コミットメッセージは変更の意図が分かるように書いてください

## 開発環境のセットアップ

```bash
git clone https://github.com/marimo-marine23/xlmelt.git
cd xlmelt
pip install -e ".[dev]"
pytest tests/ -v
```
